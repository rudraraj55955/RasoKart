# RasoKart UPI Collection Landing Page

Static responsive landing page inspired by the supplied reference structure, rebuilt with original RasoKart branding.

## Files
- `index.html` - page markup
- `styles.css` - full responsive dark SaaS design
- `script.js` - mobile menu and copy sample button

## How to use
Upload these three files to your frontend/public folder, or open `index.html` directly in a browser.

## Branding rule
This page intentionally does not show third-party provider names. Keep provider names inside Super Admin settings only.
